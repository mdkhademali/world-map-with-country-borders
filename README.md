## World Map Plot with GeoPandas

This project uses Python and GeoPandas to plot a world map with country borders.

## Requirements

- Python 3.x
- GeoPandas
- Matplotlib

Install required libraries via pip:

```bash
pip install geopandas matplotlib
```
